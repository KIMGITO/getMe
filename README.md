# 🚀 getMe

## Description

**- getMe** is a local on-demand logistics and errand platform that connects customers with nearby riders for fast, secure, and transparent delivery of groceries, parcels, and market items. It solves everyday urban delivery challenges by introducing real-time tracking, escrow payments, and verified rider workflows. The system is designed for scalability across Kenyan cities and future expansion.

---

## ✨ Features

- 🛒 Create shopping and errand requests in seconds
- 📍 Live GPS tracking of riders in real time
- 🔐 Escrow based payment system for secure transactions
- 📦 Proof of purchase via receipt image uploads
- 🔢 Handshake PIN delivery confirmation system
- 🚴 Rider duty mode and nearby job matching
- 💬 Real-time notifications and order updates
- 🧾 Order history and transaction tracking
- 🛑 Dispute management and admin oversight
- 💰 Rider wallet and earnings tracking

---

## 🛠 Tech Stack

- **Backend:** Laravel 13 (REST API, business logic, authentication)
- **Frontend (Web):** React + Vite + Tailwind CSS + ShadCN UI
- **Mobile App:** Flutter (Android & iOS)
- **Real-Time Communication:** Laravel Reverb (WebSockets)
- **Database:** MySQL with spatial indexing
- **Payments:** Safaricom M-Pesa Daraja API
- **Cloud Storage:** Receipt and document storage (S3 / Cloudinary)
- **Notifications:** Firebase Cloud Messaging (FCM)
